Markdown的简单使用

本属 2022-07-27 11:27:22
Categories： Tags：

Markdown基础用法

使用Typora进入Markdown源代码编辑模式，使用Ctrl+/即可进入

官网网址：http://www.typora.io/

可以下载排字模板和编辑主题。

标题

#+空格+标题名称———–一级标题
##+空格+标题名称———二级标题
###+空格+标题名称——–三级标题
####+空格+标题名称——四级标题
#####+空格+标题名称——五级标题
######+空格+标题名称—–六级标题
最大支持六级标题

字体

对于标准的markdown文本，是不支持居中对齐的。不过markdown支持HTML语言，所以我们采用HTML语法格式书写即可。

引用

‘>’+加空格+文字

分割线

1.三个*然后回车
2.三个-（减号）回车

插入图片

①本地的图片：英文的！+[图片名称]+（路径）
②网络图：英文的！+[图片名称]+（网络图片地址）
比如老婆帅照：

链接

[点击跳转到+名称]（网站、文件等地址）
比如：我的博客

列表

①有序列表；序号（1，①）+英文的句号（.）+输入得内容，然后回车自动补出下一个序号
②无序列表；减号（-）+空格+输入内容 回车自动补出下一个

表格

右击创建表格

  • ------: 为右对齐。
  • :------ 为左对齐。
  • :------: 为居中对齐。
  • ------- 为使用默认居中对齐。

代码

Tab键上面的点（·）三个 + 代码类型（java c++ c#等）然后回车

行列代码：Hello, this is a inline code.

数学公式

$$ + 换行 + 换行 + $$
$$
\sum_{i=0}^n i^2 = \frac{(n^2+n)(2n+1)}{6}
$$
LaTeX公式格式

Markdown快捷键

撤销：Ctrl/Command + Z
重做：Ctrl/Command + Y
加粗：Ctrl/Command + B
斜体：Ctrl/Command + I
标题：Ctrl/Command + Shift + H
无序列表：Ctrl/Command + Shift + U
有序列表：Ctrl/Command + Shift + O
检查列表：Ctrl/Command + Shift + C
插入代码：Ctrl/Command + Shift + K
插入链接：Ctrl/Command + Shift + L
插入图片：Ctrl/Command + Shift + G
查找：Ctrl/Command + F
替换：Ctrl/Command + G
